## Klipper Configs

Both my configs use CAN Bus, Fly SHT36v3 toolhead board, sensorless homing, resonance tuning scripts and git backup.

- [Voron 2.4 250mm v2.1906 config](https://github.com/richardjm/voronpi-klipper-backup) - also configures WLED macros via a separate ESP8266 board
- [Crucible (printer for ants) config](https://github.com/richardjm/crucible-klipper-backup) - for my [Crucible](../crucible.md) - with Fly Puppet 5 controller board
